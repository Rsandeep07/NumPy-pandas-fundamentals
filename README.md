# Python Data Analysis Fundamentals

This repository contains hands-on Jupyter notebooks focused on building a strong foundation in data analysis using Python.  
It covers core concepts required for Data Analyst and Data Science roles, with practical examples and clean code.

---

## Contents

- NumPy basics and array operations  
- Pandas Series creation and manipulation  
- Indexing, slicing, and filtering data  
- Basic aggregations and descriptive statistics  
- Handling missing values  
- Data inspection using head(), info(), shape(), and describe()

---

## Notebooks

- **NumPy_Pandas_Series.ipynb**  
  Covers NumPy array fundamentals and Pandas Series operations with examples.

---

## Tools & Libraries

- Python  
- NumPy  
- Pandas  
- Jupyter Notebook  

---

## Purpose

The goal of this repository is to demonstrate practical understanding of Python-based data handling and analysis fundamentals through executable notebooks.

---

## How to Use

1. Clone the repository  
2. Open the notebook using Jupyter Notebook or JupyterLab  
3. Run cells sequentially to view outputs and understand the workflow
